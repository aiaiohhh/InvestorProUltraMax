## Video Requirements

### Video Files
- **Format**: MP4 (H.264 codec recommended for best browser compatibility)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Aspect Ratio**: 16:9
- **Duration**: 1-3 minutes recommended
- **File Size**: Optimize for web (aim for < 10MB per video)

### Thumbnail Images
- **Format**: JPG or PNG
- **Resolution**: 640x360 (16:9 aspect ratio)
- **File Size**: < 200KB each

## Video Topics

1. **dashboard-overview.mp4** - Showcase the main dashboard interface, portfolio overview, and key metrics
2. **ai-insights.mp4** - Demonstrate AI-powered analysis features and insights
3. **fair-value.mp4** - Explain fair value calculations and how they help with investment decisions
4. **real-time-alerts.mp4** - Show how alerts work and real-time notifications
5. **portfolio-analytics.mp4** - Deep dive into advanced analytics and reporting features

## Tips for Creating Videos

- Use screen recording software (OBS, QuickTime, Camtasia, etc.)
- Record in a clean, professional environment
- Add subtle background music (optional)
- Keep videos concise and focused on key features
- Use smooth transitions and animations
- Ensure text is readable at different screen sizes

## Alternative: Using Placeholder Videos

If you don't have videos yet, you can:
1. Use placeholder services like [Placeholder.com](https://placeholder.com) for thumbnails
2. Create animated mockups using tools like After Effects or Figma
3. Use stock video footage from services like Pexels or Pixabay (with proper licensing)

## Testing

After adding videos, test:
- Video playback on different browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsiveness
- Loading performance
- Video controls functionality
